Locaydo is a mobile application that aims to organize local buying and selling within the Gaza Strip through a simple and structured platform, reducing the randomness currently found on social media marketplaces.

This project is developed as a **Software Engineering Graduation Project**, focusing on building a realistic solution that fits local conditions in terms of connectivity, usability, and user capabilities.

---

## ğŸ“Œ Project Idea

Currently, many people in Gaza rely on Facebook groups and social media platforms to buy and sell products. This approach leads to several issues:

* Difficulty finding relevant products
* Lack of proper categorization
* Limited trust between buyers and sellers

**Locaydo** addresses these issues by providing a single platform that offers:

* Categorized products
* Clear seller profiles
* A simple rating system
* Direct communication without unnecessary complexity
